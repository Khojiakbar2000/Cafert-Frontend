## ğŸ› ï¸ Customization Guide

### Changing Content
- Edit the HTML files in the `Template/` folder
- Update text, images, and links to match your business
- Modify contact information in `contacts.html` and `reservations.html`

### Styling
- CSS files are in `Template/css/`
- Source SCSS files are in `Template/scss/` (if you want to modify styles)
- Main color scheme and fonts can be customized

### Images
- Replace images in `Template/img/` folder
- Update image paths in HTML files
- Recommended image sizes are maintained in the template

### Contact Forms
- PHP files (`contacts.php`, `booking.php`) handle form submissions
- Configure email settings in these files
- Test forms work with your server setup

## ğŸš€ Deployment
